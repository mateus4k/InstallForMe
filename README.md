An installer of my usual programs coded in bash.

# How to use
Clone this repository:
`git chone https://github.com/mateus4k/InstallForMe`

Go to directory:
`cd InstallForMe/`

Grant full permission to script:
`chmod 777 installforme.sh`

Run!
`./installforme`

Enjoy! :+1:
